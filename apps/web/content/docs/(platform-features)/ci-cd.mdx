Subvima is perfect for preview environments in CI/CD pipelines. By using API keys, you can automate tunnel creation without interactive login.

## Setup

1.  **Generate API Key**: Go to your organization settings in the dashboard and generate a new API Key.
2.  **Store Secret**: Store the API key as a secret in your CI provider (e.g., `SUBVIMA_API_KEY`).

## Usage

Use the `--key` flag to authenticate the tunnel:

```bash
Subvima 3000 --key $SUBVIMA_API_KEY
```

## Example: GitHub Actions

Here is a simple example of how to use Subvima in a GitHub Action:

```yaml
steps:
  - name: Start Tunnel
    run: |
      npm install -g Subvima
      Subvima 3000 --key ${{ secrets.SUBVIMA_API_KEY }} &
```

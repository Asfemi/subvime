# Installation

You can install the Subvima CLI globally using npm. This is the recommended way to install Subvima as it allows you to run the `Subvima` command from anywhere in your terminal.

## Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher)
- npm (comes with Node.js)

## Install via npm

Run the following command in your terminal:

```bash
npm install -g Subvima
```

## Verify Installation

To verify that Subvima has been installed successfully, run:

```bash
Subvima --version
```

You should see the version number of the installed CLI.

## Updating

To update Subvima to the latest version, simply run the install command again:

```bash
npm install -g Subvima@latest
```

## Uninstalling

If you need to remove Subvima from your system:

```bash
npm uninstall -g Subvima
```
